Описание значимых атрибутов

Проанализировав полученную базу, могу выдвинуть предположение, что наиболее значимыми атрибутами являются полям: Название вакансии, Заработная плата, Дата публикации, Опыт работы, Ключевые навыки.

Краткое пояснение выбранных атрибутов:

  • Название вакансии - позволит сортировать или искать нужную вакансию по названию.
  • Заработная плата - показывает значимую информацию для пользователя при выборе профессии.
  • Дата публикации - позволит сортировать вакансии по свежести.
  • Опыт работы - позволит группировать вакансии, а также показывает необходимые требования пользователю при выборе профессии.
  • Ключевые навыки - показывает пользователю почти самую важную информацию при поиске професии.


    • Описание структуры набора данных

    Краткое описание каждого атрибута:

  • Название вакансии - отображает название нашей вакансии.
  • Описание вакансии - отображает основную информацию, описывающую вакансию.
  • Заработная плата - отображает сколько работник будет получать при устройстве.
  • Дата публикации - отображает, когда была вылажена вакансия.
  • Название региона - отображает регион, в котором была выложена вакансия.
  • Работодатель - отображает название работодателя.
  • Ключевые навыки - отображает ключевые навыки, необходимые при устройстве на работу.
  • Опыт работы - отображает минимальный срок опыта работы для устройство на вакансию.
  • График работы - отображает график работы вакансии.
